# 🎮 Color Filter Pro - Warzone Edition

Une application Python complète et optimisée pour améliorer les visuels de Warzone et autres jeux FPS avec des filtres de couleur avancés en temps réel.

## 🚀 Démarrage rapide

### Installation

```bash
# Cloner ou télécharger le repo
git clone https://github.com/xalith02/couleur-warzone.git
cd couleur-warzone

# Installer les dépendances
pip install -r requirements.txt

# Lancer l'application
python color_filter_warzone.py
```

## ✨ Fonctionnalités

### 🎨 9 Filtres Ajustables
- **Contraste** (0.5x à 3.0x) - Augmente la séparation entre les éléments
- **Luminosité** (0.5x à 2.0x) - Ajuste la clarté générale
- **Saturation** (0 à 2.5x) - Intensité des couleurs
- **Vibrance** (0 à 2.0x) - Saturation sélective des couleurs
- **Boost Couleur** (0.5x à 3.0x) - Amplifie l'intensité des teintes
- **Netteté** (0 à 2.5x) - Rend les détails plus tranchants
- **Flou** (0 à 10) - Réduit le bruit optionnel
- **Gamma** (0.5 à 2.5) - Correction de l'exposition
- **Décalage Teinte** (-180 à +180) - Modifie la teinte générale

## 🎮 Comment Utiliser

1. **Lancer l'application**
   ```bash
   python color_filter_warzone.py
   ```

2. **Démarrer la capture**
   - Cliquez sur "▶ DÉMARRER CAPTURE"
   - Acceptez l'accès à votre webcam

3. **Ajuster les filtres**
   - Utilisez les sliders pour ajuster les paramètres
   - L'aperçu se met à jour en temps réel

4. **Utiliser les presets**
   - Cliquez sur un preset pour appliquer une configuration optimisée
   - Ajustez ensuite les valeurs selon vos préférences

5. **Réinitialiser**
   - Cliquez sur "🔄 RÉINITIALISER" pour revenir aux valeurs par défaut

## ⚙️ Configuration Système

- **OS**: Windows 10/11, macOS, Linux
- **Python**: 3.7+
- **Webcam**: Requise (intégrée ou USB)
- **RAM**: 2GB minimum
- **Processeur**: Dual-core minimum

## 🎯 Tips & Tricks

1. **Pour les yeux sensibles**: Utilisez le preset HDR, il est moins agressif
2. **Pour plus de FPS dans le jeu**: Baissez la netteté, gardez le contraste
3. **Pour bien viser**: Augmentez le contraste et la saturation
4. **Pour l'ambiance**: Utilisez le preset Cinématique
